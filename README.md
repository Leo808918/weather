# 个人日志笔记网站

一个简洁的个人日志笔记应用，支持按日期记录笔记，并集成通义千问AI对话功能。

## 功能特性

- 📅 **日期日志**：按日期创建和管理个人日志
- ✏️ **笔记管理**：支持创建、编辑、删除日志
- 🤖 **AI对话**：集成通义千问大模型，可与AI进行对话
- 💾 **本地存储**：数据保存在浏览器本地，无需服务器
- 🔒 **安全设计**：API Key 从系统环境变量读取，不暴露在前端

## 使用方法

### 1. 启动本地服务器

在终端中运行：

```bash
cd C:\Users\Surface\Desktop\weather
python server.py
```

你会看到类似这样的输出：
```
==================================================
📝 个人日志笔记 - 本地服务器
==================================================
✅ API Key 已配置: sk-xxxxx****xxxx
--------------------------------------------------
🌐 服务器地址: http://localhost:8000
📂 静态文件目录: C:\Users\Surface\Desktop\weather
--------------------------------------------------
按 Ctrl+C 停止服务器
==================================================
```

### 2. 打开网站

在浏览器中访问：**http://localhost:8000**

### 3. 开始使用

- 点击「+ 新建」创建日志
- 点击右上角笑脸图标打开 AI 助手
- 点击设置图标查看 API 状态

## 环境变量配置

本项目通过系统环境变量 `DASHSCOPE_API_KEY` 读取 API Key，确保你已正确配置：

**Windows PowerShell (临时)**:
```powershell
$env:DASHSCOPE_API_KEY="你的API Key"
python server.py
```

**Windows 永久设置**:
1. 右键「此电脑」→「属性」→「高级系统设置」
2. 点击「环境变量」
3. 在「用户变量」中新建变量名 `DASHSCOPE_API_KEY`
